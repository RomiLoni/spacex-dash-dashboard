# SpaceX Dash Dashboard

Interactive dashboard built using Plotly Dash to analyse SpaceX launch records.

## Features
- Launch site dropdown filter
- Success vs failure pie chart
- Payload range slider
- Scatter plot showing correlation between payload mass and launch success
- Booster version performance comparison

## Technologies Used
- Python
- Pandas
- Plotly Dash
- Plotly Express

## How to Run
```bash
pip install dash pandas plotly
python spacex-dash-app.py
